# AtavismEditor

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 8.3.17.

Also as main technology is used [Electron](https://electronjs.org/) version 7.0.0

Original Theme [Fuse](http://angular-material.fusetheme.com/apps/dashboards/project)

## Development server

Dla zwyklej pracy tylko z angularem odpalamy projekt `ng serve` albo `yarn ng:serve`.

Dla pracy z aplikacją w przegłądarce `yarn ng:serve:web`.

Dla pracy z aplikacją w oknie natywnym, odpalamy projekt angulara `yarn ng:serve:web` i następnie `yarn electron:serve`

## Wysylanie zmian

Pracujemy na podstawie [Conventional Commits](https://www.conventionalcommits.org/en/v1.0.0/)

Pracujemy na oddzielnym branch-u od mastera. Nazwy zaczynają się od bugfix(albo feature) + '/' + numer taska + krótki opis.

Po zakończeniu prac robimy Pull Request, i sprawdzamy sobię kod na wzajem.
